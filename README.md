# 🏦 Customer Segmentation using K-Means

## 📌 Overview
This project applies K-Means clustering to segment banking customers based on financial and behavioral attributes.

## 🎯 Objective
Identify distinct customer segments to enable:
- Risk-based strategies
- Targeted marketing
- Portfolio optimization

## 🧠 Techniques Used
- K-Means Clustering
- Elbow Method
- Silhouette Score
- Feature Scaling

## 📊 Features Used
- Income
- Credit Limit
- Utilization Ratio
- DPD (Days Past Due)
- Number of Trades
- Average Balance

## 🚀 Results
Customers were segmented into:
- 🔴 High Risk
- 🟡 Medium Risk
- 🟢 Low Risk

## 🛠️ Tech Stack
Python, Pandas, NumPy, Scikit-learn, Matplotlib
